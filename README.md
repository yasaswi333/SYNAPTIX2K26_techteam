# SYNAPTIX2K26_techteam
Hackothon project

Smart Travel Guide Web Application

An interactive web application built using HTML, CSS, JavaScript, and Google Maps APIs that helps users discover nearby places such as restaurants, hotels, attractions, rentals, mechanics, and emergency services.

🚀 Key Features

📍 Search by place name or coordinates

🗺 Interactive Google Map with live markers

⭐ Filter by rating and place type

📊 Smart sorting (reviews, rating, images)

📏 Distance & travel time calculation

📞 Emergency contact display

📌 Current location detection

⚙️ Technologies Used

Google Maps JavaScript API

Google Places API

Distance Matrix API

Geocoding API

📂 Project Structure

index.html – Main layout

style.css – Styling

JS files – Map, search, markers, and location handling

🎯 Purpose

Designed to provide a smart, user-friendly travel assistance tool for quick and reliable location-based recommendations.
